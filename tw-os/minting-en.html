Minting village clusters calculator

This tool helps you calculate your optimal minting villages based on flags, items, building levels, et cetera.

Some things this tool takes into consideration:

  • Storage and market building levels
  • Coin cost reduction flags and flag boosters
  • Resource items that you want to use in a 48 hour period
  • Towns that can't send to a hub town fast enough will mint in it's own town if possible to prevent overflowing storage.
  • If towns can be emptied in time at multiple hubs, it will use the hub where coins are cheapest
